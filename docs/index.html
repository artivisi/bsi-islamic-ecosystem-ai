Kenapa Format Teks?

AI yang kita pakai adalah Large Language Model (LLM) — dilatih dengan teks, berpikir dalam teks, menghasilkan teks.


  • LLM membaca dan menulis teks secara native
  • File teks bisa langsung dibaca, diedit, dan ditulis oleh AI
  • File biner (DOCX, PDF, XLSX) harus dikonversi dulu ke teks
  • Semakin sedikit konversi, semakin akurat hasilnya

File Biner (Non-Teks)

PDF, DOCX, PPTX, XLSX, MP4, MP3, JPG, PNG


  • Tidak bisa dibaca langsung oleh AI sebagai teks
  • Perlu dikonversi dulu ke format teks agar bisa diolah
  • Contoh: DOCX → Markdown, XLSX → CSV, MP4 → transkrip teks
  • AI juga bisa menulis program kecil untuk membaca isi file biner

Output akhir untuk klien tetap dalam format Office

Tool yang AI Jalankan

Kebutuhan Tool Hasil untuk Anda

Semua tool dijalankan AI otomatis — kita cukup minta hasilnya.

AI Juga Bisa Menulis Program

Kebutuhan Yang AI Lakukan
Baca & analisis daftar prospek di Excel Tulis program untuk buka file, hitung statistik, urutkan
Ambil data dari portal Kemenag / BPS / dst Tulis program untuk buka web, ekstrak data, simpan ke CSV
Buat grafik distribusi prospek per segmen / provinsi Tulis program untuk render chart, simpan ke PNG/PDF
Rapikan ratusan file brief lama Tulis program untuk rename sesuai pola, pindah ke folder kategori
Hitung potensi pasar per segmen (rata-rata, total, sebaran) Tulis program statistik, jalankan, sajikan hasil

Kita tidak perlu bisa coding — AI yang menulis & menjalankan programnya.

Kombinasi Tool + Script

Contoh: Akses portal pendataan yang perlu login (Kemenag, sistem internal BSI, dst)


  • AI tulis program buka browser otomatis (Playwright)
  • Browser tampilkan halaman login, kita masukkan username & password
  • Program simpan sesi login
  • Selanjutnya AI bisa akses halaman-halaman yang dilindungi
  • AI baca, kompilasi, dan rangkum isi halaman tersebut

Teknik Tambahan: JSON sebagai Database Mini

Kenapa JSON?
  • 1 file per prospek
  • Bisa simpan struktur bertingkat (banyak kontak, banyak unit usaha, banyak isu)
  • Claude bisa search & filter di seluruh folder
  • Database pribadi, tumbuh bersama pekerjaan

  "nama": "Pesantren Sidogiri",
  "santri": 7500,
  "unit_usaha": [
    "BPRS", "Koperasi", "BMT"
  ],
  "isu_terkini": [
    "rencana RS pesantren"
  ],
  "last_visit": "2026-03-15"
}

Teknik Tambahan: Playwright untuk Situs Sulit

Untuk situs yang sulit diakses cara biasa:

  • SPA — konten baru muncul setelah JavaScript jalan
  • Portal yang wajib login (Kemenag, sistem internal, dll)
  • Situs anti-crawler yang blokir bot biasa

AI menjalankan browser sungguhan otomatis. Kita login manual sekali — sesi tersimpan, AI lanjut ambil data sendiri.
